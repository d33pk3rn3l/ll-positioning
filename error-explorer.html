BACK-OF-ENVELOPE PRINCIPLES

Anchored dead reckoning (two-filter smoother)

Forward filter chains all past fixes: propagate DR variance between anchors, fuse at each anchor; backward filter does the same over future fixes. The two use disjoint DR segments → independent:

propagate: v ← v + (k·Δd)²   fuse: 1/v ← 1/v + 1/σ²_ctrl

σ²(s) = v_f·v_b / (v_f + v_b)

No forward anchor ⇒ σ = σ_f (open traverse). k per axis: along-track = odometer scale-factor error [%·d]; cross-track = azimuth error ψ [rad]·d; up = height-transfer error [%·d].

Datum tie (local → ME frame)

σ_ME = √( σ²_local + σ²_datum )

σ_datum ≈ 10 m horiz / 2 m vert (lander located in LRO NAC). Affects georeferencing only — gravimetry uses relative σ_U.

Free-air sensitivity (the requirement driver)

∂g/∂h = 2g/R = 187 µGal/m

50 µGal ⇒ σ_U ≤ 0.27 m; error-budget allocation ⇒ σ_U ≈ 0.10 m (19 µGal).

Lava tube signal (∞ horizontal cylinder)

Δg = 2πG·Δρ·r² / z

r=30 m, z=60 m, Δρ=−3000 kg/m³ ⇒ ≈ −1.9 mGal peak (S/N ≳ 40).

Max horizontal gradient (3√3/8)·Δg/z ≈ 16 µGal/m ⇒ σ_horiz ~1–2 m suffices.
